## :computer: Setup

To run this project follow these steps:

1. Clone the repository

```
$ git clone https://github.com/kubacarbon/TicTacToe.git
```

2. Open the project

```
$ cd TicTacToe
```

3. Install dependencies

```
flutter pub get
```

4. Run app

```
$ flutter run
```
